## GitHub Pagesへようこそ！ あなたのウェブサイトのコンテンツをMarkdownで維持・プレビューするために、[GitHub上のエディター](https://github.com/eggplants/leap-day-correct-anchored/edit/gh-pages/index.md)が利用可能です。 このリポジトリにコミットするたびに、GitHub Pagesは[Jekyll](https://jekyllrb.com/)を実行し、あなたのMarkdownファイルのコンテンツから、あなたのサイトのページを再構築します。 ### Markdown Markdownは、あなたの文章をスタイリングするための軽量で使いやすい構文です。これには次のような規則があります: ```markdown Syntax highlighted code block # Header 1 ## Header 2 ### Header 3 - Bulleted - List 1. Numbered 2. List **Bold** and _Italic_ and `Code` text [Link](url) and ![Image](src) ``` 詳しくは、[基本的な書き方・書式の構文](https://docs.github.com/en/github/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax)をご覧ください。 ### Jekyllのテーマ あなたのページのサイトは、[リポジトリ設定](https://github.com/eggplants/leap-day-correct-anchored/settings/pages)で選択したJekyllテーマからレイアウトとスタイルを使用します。このテーマの名前は、Jekyllの設定ファイル `_config.yml` に保存されています。 ### サポート・連絡先 Pagesでお困りですか? [ドキュメンテーション](https://docs.github.com/categories/github-pages-basics/)を確認するか、[サポートにご連絡ください](https://support.github.com/contact)。私たちが力になります。
